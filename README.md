# Lead Sales Management System

Full-stack CRM system built with:

- Frontend: Next.js
- Backend: FastAPI
- Database: Supabase

## Features
- Lead Management
- Sales Pipeline
- Follow-ups
- Team Management

## Setup

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn main:app --reload
